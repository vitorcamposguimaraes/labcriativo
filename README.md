# Rock in Rio Brand Experience Advanced

Site estático em HTML/CSS pronto para GitHub Pages.

## Publicação

1. Crie um repositório no GitHub.
2. Envie todos os arquivos desta pasta.
3. Vá em **Settings > Pages**.
4. Em **Build and deployment**, escolha **Deploy from branch**.
5. Selecione a branch **main** e a pasta **/**.
6. Aguarde o link ser gerado.

## Estrutura

- `index.html`
- `style.css`
- `images/hero-festival.svg`
- `images/estacio-stand-concept.svg`

## Observação

As ilustrações SVG são originais e podem ser substituídas por fotos oficiais, logos ou imagens licenciadas conforme sua apresentação evoluir.
